# Frontend Foundation – Alfa Delfa

Goal:
- Black, ultra-modern dashboard for research labs.
- Start with layout, routing, and theme only (no real data).
- Tech: Next.js (app router) + React + TypeScript + Tailwind + shadcn-ui (or similar).

Initial pages:
- / (redirect to /projects)
- /projects
- /projects/[projectId]
- /runs/[runId]

Layout:
- Persistent left sidebar (logo, navigation).
- Main content area for pages.
- Right-hand panel reserved for AI copilot (for now: placeholder).

Theme:
- Full dark/black base.
- Primary accent color (e.g. electric cyan).
- Minimal color palette, high contrast, scientific/data vibe.
